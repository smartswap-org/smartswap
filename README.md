## Quick Start

1. **Clone the repository**
```bash
git clone https://github.com/smartswap-org/smartswap --recursive
cd smartswap
```

2. **Setup environment**
```bash
cp .env.example .env
```

3. **Launch with Docker**
```bash
docker-compose up
```
